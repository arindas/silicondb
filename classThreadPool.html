Detailed Description

Class managing a pool of worker threads. Work can be queued by passing a function to queue(). A future will be returned that can be used to obtain the result of the function after execution.

Usage example:

ThreadPool pool(10);
std::vector< std::future< int > > results;
for (int i=0;i<10;i++)
{
auto run = [](int i) { return i*i; };
results.emplace_back(pool.queue(std::bind(run,i)));
}
for (auto &f : results)
{
printf("Result %d:\n", f.get());
}
