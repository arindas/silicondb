Detailed Description

template<uint8_t n, typename F, typename... Ts>
struct details::HelperRecT< n, F, Ts... >

Recursive template definition for multiple arguments. Each function checks for a matching index, and if found does the action. If not, the same function is called with the next index and one less type argument.
